Construcción de una página web HTML:
  1. Crear documentos estructurados con etiquetas
  2. Elegir el mejor editor para un caso concreto de creación
  3. Crear la estructura básica de tu página HTML: cabecera y el cuerpo
  4. Crear el contenido y aplicarle formato con las etiquetas HTML
  5. Integrar elementos interactivos si es necesario
  6. Implementar la lógica para responder a dichos formularios en el caso de que se integren
  7. Publicar la página en Internet o en local
